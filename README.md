## 项目说明

本项目的主要内容包括：

- 电池健康状态（State of Health，SOH）估计
- 电池剩余寿命（Remaining Useful Life，RUL）估计

## 环境配置

使用 [miniconda](https://docs.conda.io/en/latest/miniconda.html) 进行环境配置。

配置说明和配置文件稍后更新。

## 数据集

本项目使用的电池数据包括：

- 马里兰大学电池老化数据集
- NASA电池老化数据
- Stanford商用18650电池老化数据集
